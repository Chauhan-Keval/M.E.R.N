TABLE

What is Table ?

An HTML table is used to display data in a row and column format. Tables are created using the (table) tag. Inside a table, rows are defined using (tr) table row, columns or headers using (th) table header, and data cells using (td) table data.

Table Structure

Here is a basic structure of an HTML table:

table : Defines the table.

th : Defines a table header cell (bold and centered by default).

tr : Defines a table row.

td : Defines a table data cell.

Attribute that can be use with TABLE

Here are some common attributes that can be used with the (table) tag:

border :Adds a border to table.

colspan :Used to merge Coloum.

rowspan :Used to merge Row.
